# NotebookLMプロンプトテンプレート

このテンプレートを使って `notebooklm-prompt.md` を作成します。

```markdown
# NotebookLMプロンプト

あなたは学習コーチ兼出題設計者です。
アップロードされた資料だけを根拠にして、次のテーマの学習を支援してください。

## 学習テーマ

{{TOPIC}}

## 学習目的

{{PURPOSE}}

## 学習の深さ

{{DEPTH}}

## 依頼

1. まず、このテーマの学習ロードマップを作ってください。
2. 初学者が最初に理解すべき概念を、順序立てて説明してください。
3. 重要用語を表形式で整理してください。
4. 誤解しやすいポイントを列挙し、正しい理解を説明してください。
5. 実務または応用場面での使われ方を説明してください。
6. 理解度確認クイズを作ってください。

## クイズ要件

- 用語確認問題
- 比較問題
- 引っかけ問題
- 実務シナリオ問題
- 解答と解説
- 根拠になった資料名または引用箇所

## 制約

- アップロードされた資料に基づいて回答してください。
- 資料にないことは推測せず、「資料からは判断できない」と明記してください。
- 古い情報や不確かな情報がある場合は注意点として分けてください。
```
